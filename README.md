# Bullshark Battlegrounds

A Trading Card Game built on Sui.

# Running on local machine

`git clone https://github.com/RevanthGundala/Bullshark-Battlegrounds.git`

# Running integration tests

1. `cd card_game`
2. `sui move test`

# Backend

1. `cd backend`
2. `npm run dev`

# Frontend

1. `cd frontend`
2. `npm run dev`
